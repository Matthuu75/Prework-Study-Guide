HTML

  • HTML = Hyper Text Markup Language
  • The head element contains information, or metadata, about the webpage
  • Metadata = data about our webpage that the browser uses to effectively serve our webpage to our users
  • The body element represents the visible content shown to the user
  • HTML elements are case sensitive
  • HTML Attribute - defines additional information about and element (IMPORTANT: Always defined in the beginning of the element, NEVER the closing element)
  • Child elements are elements nested inside of other elements
  • Parent elements are elements that hold all the child elements
  • Openeing tag and closing tags are represented by angle brackets
  • Closing tags begin with a forward slash

CSS

  • A margin indicates how much space we want around the outside of any element
  • A padding indicates how much space we want around the content inside an element

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
